    Class BackendApiService

    This Angular service provides the methods for all requests from a caller to the Lonely Lobster backend. It makes the call to the backend and returns an Observable through which the requested response will be delivered to the caller. In case the Observable returns an error it first calls an error handler which then signals an error back to the caller.
